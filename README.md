## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Web3 wallet (MetaMask recommended)
- OG tokens for transactions

### Installation
```bash
# Clone the repository
git clone https://github.com/ayushsingh82/OG-Data.git

# Install dependencies
cd AgentForge
npm install

# Start development server
npm run dev
```

### Smart Contract Deployment
```bash
# Navigate to contracts directory
cd contract

# Install dependencies
npm install

# Compile contracts
npx hardhat compile

# Deploy to 0G network
npx hardhat run scripts/deploy.js --network 0g-testnet
```
